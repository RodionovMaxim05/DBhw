# Homework for course "Databases"

## Требования
### Продажа автомобилей
Задача заключается в автоматизации работы компании, которая занимается продажей новых и подержанных автомобилей.
В системе должен вестись учет продаж автомобилей, причем должны фиксироваться не только модели и года выпуска, но и другие базовые характеристики новых и подержанных автомобилей, а для последних – еще и показатели их состояния.
При учете продаж следует автоматически отслеживать рейтинг сотрудников компании (в первую очередь - количество продаж), поскольку эти данные впоследствии будут передаваться в отдельную расчетную систему.
Кроме того, система должна хранить информацию для создания статистических отчетов по клиентам (физическим и юридическим лицам).

## Структура проекта
```
.
├── create_tables.sql        # Создание таблиц и справочников
├── drop_objects.sql         # Скрипт удаления всех объектов БД
├── functions.sql            # Функции (статистика, отчёты в JSON)
├── procedures.sql           # Хранимые процедуры (резервирование, возврат продажи)
├── sample_data.sql          # Наполнение базы тестовыми данными
├── triggers.sql             # Триггеры (автообновление статусов, статистики и т.д.)
├── view_all_tables.sql      # Просмотр текущего состояния всех таблиц
├── selects.sql              # SELECT-запросы для отчётов и интерфейса
└── views.sql                # Представления для UI и отчётов
```
